## Features

- **AXON Binary IDL** вҖ” Zero-copy binary serialization, no JSON overhead
- **Pub/Sub + RPC + Actions** вҖ” Full messaging primitives
- **AI Router** вҖ” Plug GPT, Claude, or any provider by API key
- **Robot Connector** вҖ” HTTP/WebSocket interface for any hardware
- **Luxury Dashboard** вҖ” Real-time node graph, topic monitor, AI console
- **ROS 2 Bridge** вҖ” Gradual migration support
- **Sub-millisecond local transport** вҖ” Shared memory ring buffers

## Quick Start

```bash
# Clone
git clone https://github.com/phoenixc13/apex-middleware.git
cd apex-middleware

# Install dependencies
pip install -r requirements.txt

# Configure your AI providers
cp config/providers.example.yaml config/providers.yaml
# Edit config/providers.yaml with your API keys

# Run APEX
python apex/main.py

# Open Dashboard
# http://localhost:8000
```

## AI Provider Configuration

```yaml
# config/providers.yaml
providers:
  build_ai:
    name: openai
    model: gpt-4o
    api_key: YOUR_OPENAI_KEY
    role: code_generation

  runtime_ai:
    name: anthropic
    model: claude-opus-4-5
    api_key: YOUR_ANTHROPIC_KEY
    role: task_planning

  robot_ai:
    name: openai
    model: gpt-4o-mini
    api_key: YOUR_OPENAI_KEY
    role: robot_commands
```
